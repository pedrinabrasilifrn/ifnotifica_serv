IF-Notifica

O aplicativo e sistema de dashboard do IF-Notifica foi criado com o objetivo registrar e diminuir o tempo levado para o envio de notificacoes da Covid-19 no municipio de São Paulo do Potengi e posteriormente para outras cidades!

Acesse a dashboard baixar aplicativo

Justificativa

Com o surgimento da pandemia da Covid-19, o mundo se viu diante de uma crise sem precedentes, e com isso, a necessidade de se criar mecanismos para o controle e monitoramento da doenca. Um destes mecanismos eram as notificacoes, que sao enviadas para o E-sus Notifica, um sistema do governo federal que tem como objetivo monitorar a pandemia no Brasil. Entretanto, o envio das notificacoes era um processo demorado, logo que o envio de uma notificacao para o E-sus poderia levar até 10 minutos, tendo em vista que a quantidade de notificoes enviadas diaramente passava de 80 durante a pandemia.

Objetivos

Assim sendo, decidimos criar um aplicativo que iria otimizar o tempo gasto para criacao e envio das notificacoes, e posteriormente desenvolvemos uma dashboard que iria mostrar os dados contidos nas notificacoes de forma mais intuitiva e interativa. Tendo como meta:

  • Mapear o processo de notificação dos casos de COVID-19 nos sistemas SIVEP-Gripe e e-SUS Notifica junto a secretaria de saúde do município e identificar os entraves do processo de notificação dos casos da COVID-19 no município de São Paulo do Potengi
  • Desenvolver um aplicativo para dispositivos móveis que possibilite a notificação dos casos de COVID-19 no município de São Paulo do Potengi
  • Desenvolver uma dashboard que possibilite a visualização dos dados contidos nas notificações de forma mais intuitiva e interativa

Aplicativo

Aplicativo criado para optimizar o tempo levado para a criação de uma noticação a partir do uso de tecnologia de APIs externas.

API

API responsavel por realizar a comunicação dos dados enviados pelo aplicativo para nossa dashboard e para o E-SUS Notifica.

Dashboard

Dashboard desenvolvida com o intuito de trazer uma forma acessivel de vizualizar os dados enviados pelo APP.

Offline

Visando acessibilidade, o aplicativo armazena todos seus dados nativamente, para serem enviados quando o usuario tiver acesso a rede.

Sicronizado

Sempre que uma notificação é enviada, a API e os graficos da Dashboard são atualizadas automaticamente.

Data Viz

A dashboard possui diversas metricas criadas para monitorar o avanço da covid-19.

Resultados alcançados

Tendo em vista os objetivos propostos anteriormente, conseguimos alcancar os seguintes resultados:

  • Mapeamento e identificação dos entraves do sistema de notificação em São Paulo do Potengi.
  • Diminuição do tempo realizado para enviar uma notificação para menos de um minuto e a incidencia de erros humanos com a criação do aplicativo.
  • Melhorias na acessibilidade pelo meio off-line a partir do salvamento local de notificações no aplicativo
  • Melhorias no gerenciamento e vizualização de dados com a criação da Dashboard e API do IF-Notifica
